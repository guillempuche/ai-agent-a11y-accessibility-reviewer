# ai-agent-a11y-accessibility-reviewer

Use this agent to review code for accessibility (a11y) compliance. Use after writing UI components, forms, navigation, or interactive elements. Evaluates WCAG 2.1/2.2, WAI-ARIA, VoiceOver, and TalkBac

## Install

```bash
# Add marketplace
/plugin marketplace add guillempuche/ai-agent-a11y-accessibility-reviewer

# Install plugin
/plugin install ai-agent-a11y-accessibility-reviewer@guillempuche-ai-agent-a11y-accessibility-reviewer
```

## Part of AI Standards

This agent is also available in the [ai-standards](https://github.com/guillempuche/ai-standards) bundle with other skills and agents.
